## Mods in this repository

### ChaoticStart
Breaks up the Factions/Kingdoms for an all-out free-for-all at the start of the campaign.

### Tournament Tracker
A lightweight, read-only campaign overlay that lists every town currently hosting an active
tournament, along with the prize item and its estimated gold value.

**How to open:** Press **Ctrl + T** while on the campaign map.

See [`TournamentTracker/README.md`](TournamentTracker/README.md) for full build and installation instructions.
